To make a new test.

  1. Click on file.
  2. Select create new test.
  3. Enter a name for the test.
  4. Click add new test.

To auto load a test

  1. Click Settings and click Start Up Setting's Form.
  2. Check the box beside auto load test.
  3. Click the Add File To Auto Load List.
  4. Add as many file's as you like.
  5. After you add all your files, click the Save Settings button.
  6. To remove a file from the auto load list,
    click the red trash can beside it and click the Save Settings button.

Start Up Settings

  1. Click Settings and click Start Up Setting's Form.
  2. Pick your font size.
  3. Add as many file's as you like, to the auto load list.
  4. After you make your selection's click the Save Settings button.

Delete a line from a test.

  1. Click on the name of the test.
  2. Click on the edit test icon
  3. Hold the ctrl key and click the trash can beside the text line.
